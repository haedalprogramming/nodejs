### 🧠 Git & GitHub CLI (가장 중요 / 가장 오래 걸림)

1.  Git 설치
2.  git config 설정

``` bash
git config --global user.name "이름"
git config --global user.email "이메일"
```

3.  GitHub Repository 생성\
    → `linktree-project`

4.  Local ↔ Remote 연결

``` bash
git init
git remote add origin https://github.com/username/linktree-project.git
```

5.  첫 커밋

``` bash
git add .
git commit -m "first commit"
git push -u origin main
```

### 📩 Input & Form (UI만 구현)

``` html
<form>
  <input type="text" placeholder="이름" required>
  <input type="email" placeholder="이메일" required>
  <textarea placeholder="메시지를 입력하세요"></textarea>
```

📌 학습 포인트

-   `placeholder`
-   `required`
-   `type="email"`
-   `button type="submit"`

💬 설명 포인트:

> 지금은 껍데기(Frontend)만 만든 상태입니다.\

## 2️⃣ Meta Tags & SEO (30분)

### 📲 카톡 공유 썸네일 만들기 (OG Tag)

`<head>` 안에 추가:

``` html
<meta property="og:title" content="홍길동의 링크트리">
<meta property="og:description" content="풀스택 개발자 포트폴리오입니다.">
<meta property="og:image" content="./images/og-thumb.jpg">
```

⚠ 주의:

-   images 폴더 안에 `og-thumb.jpg` 파일이 실제로 존재해야 함
-   파일이 없으면 카톡 미리보기가 깨짐
-   수업 자료 폴더에 예제 이미지 미리 준비 권장

## 1️⃣ Git 인증 이슈 (비밀번호 중단)

2021년부터 GitHub는 비밀번호 인증을 지원하지 않습니다.

### 상황

터미널에서 비밀번호 입력 요구 → GitHub 웹사이트 비밀번호 입력 → 실패

### 대비책

**방법 A (추천)**\
VS Code → 왼쪽 Source Control 탭 → "GitHub에 게시" 버튼 사용\
→ 브라우저 인증으로 자동 연결

**방법 B (정석)**\
GitHub에서 Personal Access Token 발급 후 비밀번호 대신 입력\
→ 초보자에게는 난이도 높음

🎯 전략:

> CLI가 막히면 바로 VS Code GUI 기능으로 우회시킨다.
