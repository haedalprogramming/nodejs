# 🚀 Node.js 백엔드 속성 코스: "나만의 웹 서비스 만들기"
node.js 로 배우는 백엔드 튜토리얼. html, css, js, 부트스트랩, node까지 빠르게 웹 개발을 맛보기로 배웁니다.


## **Part 1. 기초 다지기 (The Foundation)**

웹의 겉모습을 만들고 다듬는 단계입니다.

* **01. 환경 설정:** VS Code 설치, Node.js 설치, Hello World 출력하기
* **02. HTML5:** 웹사이트의 뼈대 잡기 (태그, 구조, 폼 입력)
* **03. CSS3 & Bootstrap:** * CSS 기초 (색상, 레이아웃, 박스 모델)
  * **Bootstrap 5 활용:** 코드 한 줄로 예쁜 버튼과 네비게이션 바 만들기 (현업의 생산성 맛보기)
* **04. JavaScript 기초:** 변수, 함수, 이벤트 리스너 (버튼 클릭하면 알림창 띄우기)

---

## **Part 2. 서버 구축하기 (The Engine)**

이제 본격적으로 '백엔드'의 세계로 들어갑니다.

* **05. Node.js와 Express:** 왜 Express를 쓰는가? 서버 객체 생성과 포트 열기
* **06. 라우팅(Routing):** 사용자가 `/` 혹은 `/about`으로 접속했을 때 각각 다른 화면 보여주기
* **07. 미들웨어와 정적 파일:** HTML, CSS 파일을 서버를 통해 브라우저에 전달하기

---

## **Part 3. 동적 웹페이지와 데이터 (The Logic)**

사용자와 소통하는 서버를 만듭니다.

* **08. 템플릿 엔진 (EJS):** HTML에 자바스크립트 변수 심기 (데이터에 따라 변하는 화면)
* **09. GET & POST 요청:** * 클라이언트가 서버로 데이터 보내기 (Form 데이터 처리)
  * 서버에서 데이터 받아오기
* **10. 데이터 보관하기:** 데이터베이스(DB) 대신 메모리(Array)에 임시로 데이터 저장해보기

---

### **Part 4. 실전! 미니 프로젝트 (The Project)**

앞서 배운 모든 기술을 하나로 합칩니다.

* **11. 한 줄 방명록 서비스:**
  * 방명록 작성 폼 만들기 (HTML/Bootstrap)
  * 작성한 내용 서버로 전송 (POST)
  * 서버에서 목록 업데이트 후 화면에 뿌려주기 (EJS)
* **12. 서비스 배포 맛보기:** 내 컴퓨터가 아닌 인터넷 세상에 내 웹사이트 올리기 (Render 혹은 Glitch 활용)
